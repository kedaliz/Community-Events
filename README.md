To run this application locally, you must start the backend and frontend simultaneously. Open two separate terminal windows in your Codespace or local environment:

### Terminal 1: The Backend (Server)
This handles the database connection and API routes.
1. `cd server`
2. `npm run dev`

### Terminal 2: The Frontend (Client)
This launches the user interface.
1. `cd client`
2. `npm run dev`
